FileÂ¶

Appends the messages to a file. If the file doesn’t exist it will be created.

One important configuration option:

  • path Which is the ‘root’ for where the sender will write

Example:

file = nc.FileSender()
file["path"] = "."

adr = nc.Address()
adr["protocolName"] = "file"
adr["endpoint"] = "file:///by_rest"
recv = nc.ResolverMap()
recv["pattern"] = "file@example.org"
recv["receiver"] = adr

resolv = nc.Static()
resolv["exact"] = [recv]

What this means is that if there is a message that the router decides should be sent to “file@example.org“. It will be send by the FileSender and will be written to the file “./by_rest” relative to the place from where the node was started.
