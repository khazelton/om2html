DjbmaildirÂ¶

Works on a directory that follows the http://en.wikipedia.org/wiki/Maildir specification. Each file that are place in the new directory are expected to represent an outgoing OM2 message.

The name of the file is the time when the file was created.

Typical configuration:

djb = nc.DjbMaildirSender()
djb["create"] = True
djb["root"] = "."

create because you want the sender to build the directory structure if it does not exit.

Where the directory are to be built is given by the resolver, like this:

resolv = nc.Static()
adr = nc.Address()
adr["protocolName"] = "djbmaildir"
adr["endpoint"] = "file:///incoming"
recv = nc.ResolverMap()
recv["pattern"] = ["sink@example.org"]
recv["receiver"] = adr
resolv["exact"] = [recv]
conf["resolver"] = resolv

Which means that any message to “sink@example.org” will be added to the directory structure rooted in “./incoming”.
