RestÂ¶

Does not have any configuration options of it’s own except for ‘ssl’ which should be used if you want to use HTTPS. If so the expectation is that you use both client and server certificates.

So the simples configuration would be:

rest = nc.RestSender()

resolv = nc.Static()

adr = nc.Address()
adr["protocolName"] = "rest"
adr["endpoint"] = "http://localhost:8087/"
recv = nc.ResolverMap()
recv["pattern"] = "receiver@example.org"
recv["receiver"] = adr
resolv["exact"] = recv

Which means, if the router has fick the address “receiver@example.org” as receiver of the message. Then the resolver will pick the protocol ‘rest’ and the endpoint ‘http://localhost:8087/‘ and the message will be sent by the RestSender to that URL.
