DevnullÂ¶

Writes the message to ‘/dev/null’ the bit bucket.

Configuration example:

devnull = nc.DevNullSender()

resolv = nc.Static()

adr = nc.Address()
adr["protocolName"] = "devnull"
adr["endpoint"] = "/dev/null"
recv = nc.ResolverMap()
recv["pattern"] = "error@example.org"
recv["receiver"] = adr
resolver["exact"] = recv

Which means that any message that are supposed to be transmitted to “error@example.org” will be dropped in the bit bucket by the DevNullSender.
