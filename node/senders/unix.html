Unix Domain SocketÂ¶

Sends, that is writes to a Unix domain socket.

A simples configuration could be:

unixsock = nc.UnixSocketSender()
unixsock["path"] = "/tmp/"

Coupled to:

resolv = nc.Static()
# Default receiver
adr = nc.Address()
adr["protocolName"] = "unixsocket"
adr["endpoint"] = "file:///node2"
recv = nc.ResolverMap()
recv["pattern"] = ["sink@example.com"]
recv["receiver"] = adr
resolv["exact"] = [recv]
conf["resolver"] = resolv

Will lead the sender to write messages destined for sink@example.com to be written to the unix domain socket at ‘/tmp/node2’.
